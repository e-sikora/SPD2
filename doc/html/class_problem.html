Detailed Description

template<class Item>
class Problem< Item >

A class representing a problem with a list of items.

This class provides methods to manipulate a list of items, including loading items from a file, performing various sorting algorithms, calculating work time, and saving results to a file.

Template Parameters
ItemThe type of items in the problem.
